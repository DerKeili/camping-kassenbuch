1. Gehe auf supabase.com → kostenlosen Account anlegen
2. Neues Projekt erstellen → Region: Europe (Frankfurt)
3. Settings → API → URL und anon-Key kopieren
4. SQL Editor → Setup-Script aus der .sql-Datei ausführen
